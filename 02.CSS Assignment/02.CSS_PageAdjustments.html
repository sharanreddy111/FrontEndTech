Q_02.Visibility and Display

VISIBILITY :

The visibility property specifies whether or not an element is visible.

  1. visible : Default value. The element is visible
  2. hidden : The below element is hidden but still takes up space(check list item no.)
  3. This line is hidden
  4. collapse : This value removes a row or column, but it does not affect the table layout. The space taken up by the row or column will be available for other content.
    visibility visible hidden collapse

DISPLAY :

  • CSS display is the most important property of CSS which is used to control the layout of the element. It specifies how the element is displayed.
  • Every HTML element has a default display value depending on what type of element it is. The default display value for most elements is block or inline.
  1. Examples of block-level elements : divh1-h6pformheaderfootersection

  2. Examples of inline elements :

    span,

    a,

    img.

Q_03 Margin and Padding

Difference between CSS margin and padding is CSS margin defines the space around elements where as CSS padding is affected by the background colors. It clears an area around the content.

  1. CSS Margin property is used to define the space around elements. It is completely transparent and doesn't have any background color. It clears an area around the element.

  2. CSS Padding property is used to define the space between the element content and the element border.
